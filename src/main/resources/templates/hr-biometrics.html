TCMS CSV layout (12 columns in order)

  1. User ID
  2. First name
  3. Last name
  4. Employee ID (EAC)
  5. Date (DD/MM/YYYY)
  6. In
  7. Out
  8. Short
  9. OT
  10. LeaveType
  11. Work code
  12. Day type (Workday / Restday / Holiday)
  • Save as UTF-8 CSV. First line may be a header (detected and skipped) or the first data row.
  • Delimiter: comma or semicolon. Quoted fields (e.g. names with commas) are supported.
  • At least 10 columns (through LeaveType); columns 10–11 optional (day type defaults to Workday).
  • Empty In/Out on rest/holiday rows is OK. EAC in column 4 (index 3) must match an employee’s custom_employee_id.
